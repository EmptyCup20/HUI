# HUI

海康共享视觉展示网站

全局安装`gulp`
```

npm install -g gulp
```

#### 项目依赖安装

```
  npm install
```

然后运行
```
    node app.js
```
## V1.0功能

- 前端
- 首页展示
- HUI设计语言(平台)
- 资源库展示(UIKIT ICONFONT 彩色图标)
- 作品池

- 后端
- 图标库管理
- UIKIT
- 作品池管理
